# ⚙️ Setup & Installation

## 1️⃣ Clone project

```sh
git clone https://github.com/<your-org>/vinafitai.git
cd vinafitai
```

2️⃣ Install dependencies
```
Backend (.NET)

cd backend
dotnet restore

AI Service

cd ai-service
pip install -r requirements.txt

Mobile App

cd mobile-app
npm install

Admin Dashboard

cd admin-dashboard
npm install
```
